# 🚀 Docket-Django

A smart task management web application built using Django.

---

## 📌 Features

* 🔐 User Authentication (Login/Register)
* 📋 Task Management (Add, Edit, Delete)
* 🎯 Priority-based Tasks
* 📅 Deadline Tracking
* 📊 Productivity Dashboard *(coming soon)*

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, Bootstrap
* **Database:** SQLite

---

## ⚙️ Installation

```bash
git clone https://github.com/YOUR_USERNAME/docket-django.git
cd docket-django
python -m venv venv
venv\Scripts\activate
pip install django
cd docket
python manage.py runserver
```
